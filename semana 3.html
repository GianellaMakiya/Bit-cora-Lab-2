Repaso y avance de Página Web

Definiciones generales

  • Es un espacio digital en Internet, constituido por diversos documentos (textos, fotos, videos). Estos se estructuran para dar esencia en línea.
  • Es un conjunto de páginas bien organizadas
  • Puede estar creado con código html.
  • Necesita alojamiento "hosting" y que se relacione a un dominio, que es el nombre del sitio, que le permitirá ser visisble en Internet.

    Diferencia enrte una Web Dinámica y Estática

  • WEB DINÁMICA: Es aquella página que contiene aplicaciones dentro de la propia Web, otorgando mayor interactividad con el navegante.
  • WEB ESTÁTICA: Es aquella página enfocada principalmente a mostrar una información permanente, donde el navegante se limita a obtener dicha información, sin poder interactuar con la página web visitada.

    Google Sites

    Una vez aprendido un poco sobre el uso del HTML, pasamos a utilizar otra herramienta que nos ayudará para realizar nuestro proyecto final. Es una herramienta muy intuitiva, ya sea para poner texto, cambiar de fondo, cambiar de tema, añadir imágenes, etc.

    Permite crear un sitio web de una forma tan sencilla como editar un documento. Con Google Sites, los usuarios pueden reunir, en un único lugar y de una forma rápida, información variada, como calendarios, vídeos, presentaciones, archivos de texto, hojas de cálculo, etc.

    Ventajas:

  • Mejorar el trabajo colaborativo.
  • Manejar la seguridad de la información.
  • Agilizar los procesos de desarrollo de proyecto.
  • Usar aplicaciones para mantenerse conectado.
  • Tener mayor almacenamiento para generar más contenido.
  • Crear perfiles de usuarios para administradores y editores.
