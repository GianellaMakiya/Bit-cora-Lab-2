HTML, CSS Y Visual Studio

¿Qué es?

El html es un lenguaje de marcado que define la estructura de tu contenido , una serie de elementos que usarás para encerrar diferentes partes y que se vean o comporten de una determinada manera.

!DOCTYPE html Es el tipo de documento y es un preámbulo requerido para el funcionamiento de la página web.

Etiqueta html Este elemento encierra todo el contenido de la página entera (elemento raíz).

Etiqueta body Encierra contenido que se desea mostrar al visitante de la página web (texto, imágenes, videos, sonido, etc).

Etiqueta p Etiqueta párrafo, se usa para ingresar texto. Si quieres especificar que se trata de un párrafo, podrías encerrar el texto con la etiqueta de párrafo

Etiqueta head Es un contenedor que sirve para agregar contenido no visible para los visitantes de la página.

Etiqueta title Establece el título de la página (barra del navegador).

VÍNCULOS

Los vínculos o enlaces son los que hacen de la web. Para implementar un vínculo, necesitas usar un vínculo simple (a), es la abreviatura de la palabra inglesa «anchor».
